Assignment 1 Sorting Polys


This is a program that reads a file with random 3d shapes and organizes them.

It allows the user to: 
	Sort 3D shapes in 6 different ways
	Show the speed of said sorts
	Show the new sorted output


The date is 05/10/2021


Authors are Eunji Lee, Jonghan Park, Jihoon Oh and Bushra Osman.


To run the program, start a terminal in the directory of the .jar file and enter:

   java -jar Sort.jar 

This will execute the main program.
